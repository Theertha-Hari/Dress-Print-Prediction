# Dress Print Prediction

This project classifies Indian dress prints ( Ikkat, Kalamkari, Ajrakh) from images using deep learning and object detection models. It demonstrates practical application of YOLO and computer vision for fashion pattern recognition.

---

## Features

- Detects and classifies various Indian textile prints  
- Uses YOLO-based deep learning model  
- Handles diverse input images  
- Outputs categorized prediction results  

---

## Tech Stack

- **Python**  
- **YOLO (You Only Look Once)**  
- **OpenCV**  
- **PyTorch / Deep learning framework**  
- **NumPy**  

---

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/Dress-Print-Prediction.git
cd Dress-Print-Prediction
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the main script (example):

```bash
python predict.py
```

4. The model will classify the dress print and display results.
